🎮 Impact Simulator Page (ais.html)

✅ Validation Status: PASSED - No Errors

W3C Validator screenshot showing successful validation

📝 Reflection

The Impact Simulator page had 6 validation errors initially. The main issue was div elements nested inside code tags. Since the code tag only accepts inline elements, I replaced all the div tags with span tags and added display: block; in the CSS to keep the layout unchanged. This taught me that not all elements can be freely nested — each element has a content model that restricts what is valid inside it. After these fixes, the page passed validation with no errors.

📄 Content Page (content_ST2.html)

✅ Validation Status: PASSED - No Errors

W3C Validator screenshot showing successful validation

📝 Reflection

The Content Page validation passed after fixing issues with image filenames containing spaces. I replaced spaces in filenames with hyphens to ensure proper URL encoding. I also fixed improper use of figure and figcaption elements — I changed the image containers to use proper figure/figcaption structure for images with captions. The page now has a proper heading structure (h1 to h3) and all images have appropriate alt text describing the content related to digital cost barriers.

⬅ Back to Page Editor (Content Page Section)

📋 Overall Validation Summary

All three pages I implemented (Splash Screen, Impact Simulator, Content Page) passed W3C HTML validation with no errors. The main challenges I faced and resolved were:

  • Spaces in filenames: Using hyphens instead of spaces solved this issue
  • Heading hierarchy: Ensuring proper sequential heading levels (h1 → h2 → h3)
  • Element nesting restrictions: Learning which elements can contain which others (e.g., code tags only accept inline elements)
  • Figure and figcaption usage: Using proper semantic elements for images with captions

The validation process helped me write cleaner, more semantic HTML and understand web standards better. It also improved the accessibility of my pages by ensuring proper document structure and alt text for all images.
